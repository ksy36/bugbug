## Task: Web-compatibility triage

### Your job:

Reproduce the reported issue and do not attempt to debug or perform root cause analysis.

### Procedure

1. Identify the affected URL and the described broken behavior.
2. Navigate to the URL using Firefox Devtools MCP and try to reproduce the issue.
3. Report your findings.

**Stay focused on reproduction. Avoid:**

- Investigating WHY it's broken
- Analyzing JavaScript code
- Reading source files from the website
- Proposing fixes or theories
- Checking console errors or network requests (not needed for reproduction)
